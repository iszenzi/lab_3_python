# Лабораторная работа №3 - Алгоритмический мини-пакет (структуры данных и сортировки

## Описание
Этот проект представляет собой наборосновных алгоритмов сортировки, математических функций и структур данных на Python. Все алгоритмы реализованы без использования встроенных функций сортировки Python (`list.sort()`, `sorted()`). Проект содержит бенчмарки

## Реализованные компоненты

### Математические функции

- `factorial()` - вычисление факториала числа
- `factorial_recursive()` - вычисление факториала числа (рекурсивно)
- `fibo()` - вычисление числа Фибоначчи
- `fibo_recursive()` - вычисление числа Фибоначчи (рекурсивно)

### Сортировки
- `bubble_sort()` - пузырьковая сортировка
- `quick_sort()` - быстрая сортировка
- `counting_sort()` - сортировка подсчетом
- `radix_sort()` - поразрядная сортировка
- `bucket_sort()` - блочная сортировка
- `heap_sort()` - пирамидальная сортировка

### Стек

#### Методы:
- `push()` - добавление элемента в стек
- `pop()` - извлечение верхнего элемента из стека
- `peek()` - просмотр верхнего элемента стека
- `is_empty()` - проверка на пустоту стека
- `__len__()` - количество элементов в стеке
- `min()` - минимальный элемент в стеке

### Очередь

#### Методы:
- `enqueue()` - добавление элемента в очередь
- `dequeue()` - извлечение первого элемента из очереди
- `front()` - просмотр первого элемента очереди
- `is_empty()` - проверка на пустоту очереди
- `__len__()` - количество элементов в очереди

## Запуск программы
Запуск программы осуществляется при помощи команды в терминале
```python
python -m src.main
```
