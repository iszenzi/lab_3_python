# Лабораторная работа №3 - Алгоритмический мини-пакет (структуры данных и сортировки)

## Описание
Этот проект представляет собой наборосновных алгоритмов сортировки, математических функций и структур данных на Python. Все алгоритмы реализованы без использования встроенных функций сортировки Python (`list.sort()`, `sorted()`). Проект содержит бенчмарки

## Реализованные компоненты

### Математические функции

- `factorial(n: int) -> int` — вычисление факториала (итеративно)
	- `n`: целое неотрицательное число
- `factorial_recursive(n: int) -> int` — вычисление факториала (рекурсивно)
	- `n`: целое неотрицательное число
- `fibo(n: int) -> int` — вычисление $n$-го числа Фибоначчи (итеративно)
	- `n`: целое неотрицательное число
- `fibo_recursive(n: int) -> int` — вычисление $n$-го числа Фибоначчи (рекурсивно)
	- `n`: целое неотрицательное число

### Сортировки

- `bubble_sort(a: list[int]) -> list[int]` — пузырьковая сортировка
	- `a`: список целых чисел
- `quick_sort(a: list[int]) -> list[int]` — быстрая сортировка
	- `a`: список целых чисел
- `counting_sort(a: list[int]) -> list[int]` — сортировка подсчётом
	- `a`: список целых чисел
- `radix_sort(a: list[int], base: int = 10) -> list[int]` — поразрядная сортировка
	- `a`: список целых неотрицательных чисел
	- `base`: основание системы счисления (по умолчанию `10`)
- `bucket_sort(a: list[float], buckets: int | None = None) -> list[float]` — блочная сортировка
	- `a`: список вещественных чисел в диапазоне `[0, 1)`
	- `buckets`: число корзин (если `None`, используется `len(a)`)
- `heap_sort(a: list[int]) -> list[int]` — пирамидальная сортировка
	- `a`: список целых чисел

Дополнительно (вспомогательная функция):
- `shift_down(array: list[int], start: int, end: int) -> None` — просеивание элемента вниз в куче (для `heap_sort`)
	- `array`: список, представляющий бинарную кучу
	- `start`: индекс корневого элемента для просеивания
	- `end`: индекс последнего элемента в куче

### Стек

- `Stack` — структура данных «стек»
	- `__init__()` — создание пустого стека
	- `push(x: int)` — добавить элемент в стек
		- `x`: целое число
	- `pop() -> int` — извлечь верхний элемент
	- `peek() -> int` — посмотреть верхний элемент без удаления
	- `is_empty() -> bool` — проверка на пустоту
	- `__len__() -> int` — количество элементов
	- `min() -> int` — минимальный элемент в стеке

### Очередь

- `Queue` — структура данных «очередь»
	- `__init__()` — создание пустой очереди
	- `enqueue(x: int)` — добавить элемент в конец очереди
		- `x`: целое число
	- `dequeue() -> int` — извлечь элемент из начала очереди
	- `front() -> int` — посмотреть первый элемент без удаления
	- `is_empty() -> bool` — проверка на пустоту
	- `__len__() -> int` — количество элементов

### Бенчмарк

Функции:
- `timeit_once(func, *args, **kwargs) -> float` — замер времени одного запуска функции
	- `func`: функция для измерения
	- `args`: позиционные аргументы, передаваемые в `func`
	- `kwargs`: именованные аргументы, передаваемые в `func`
- `benchmark_sorts(arrays, algos) -> dict[str, dict[str, float]]` — сравнение сортировок на наборах данных
	- `arrays`: словарь массивов для тестирования (`{name: list[int] | list[float]}`)
	- `algos`: словарь алгоритмов сортировки (`{name: callable}`)
